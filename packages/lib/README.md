# `@apex-rn/library`

This package is a library of UI componets, widgets and hooks to use for you ReactNative application for [APEX](https://demo.alphapoint.com/) platform.

## Installation

To install it run 
```
npm i @apex-rn/library
```
You need to either login into Alphapoint's Nexus registry or get a `.npmrc`.

Pay attention that you need also following native dependencies in your RN app:

* `@react-native-clipboard/clipboard`
* `@react-native-community/blur`
* `@react-native-community/netinfo`
* `@shopify/flash-list`
* `@sumsub/react-native-mobilesdk-module`
* `react-native-biometrics`
* `react-native-camera-kit`
* `react-native-gesture-handler`
* `react-native-linear-gradient`
* `react-native-mmkv`
* `react-native-permissions`
* `react-native-safe-area-context`
* `react-native-svg`
