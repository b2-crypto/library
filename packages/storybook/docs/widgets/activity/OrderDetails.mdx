## Types

```tsx
interface OrderActivity extends ActivityItem {
  /* Describes the type of order this is. */
  type: OrderType;
  /* The current state of the order. */
  status: OrderState;
  /* Order instruments icons */
  iconSymbols?: readonly [string, string];
  /* The side of a trade. */
  side: OrderSide;
  /* The average executed price for the instrument in the order. */
  avgPrice: number;
  /* Limit price of the order */
  limitPrice?: number;
}

type OrderType =
  | 'Unknown'
  | 'Market order'
  | 'Limit'
  | 'StopMarket'
  | 'StopLimit'
  | 'TrailingStopMarket'
  | 'TrailingStopLimit'
  | 'BlockTrade';

type OrderState =
  | 'Unknown'
  | 'Working'
  | 'Rejected'
  | 'Canceled'
  | 'Expired'
  | 'FullyExecuted';

type OrderSide = 'Buy' | 'Sell' | 'Short' | 'Unknown';
```

## Hooks

### `useCancelOrderMutation()`

This hook provides a wrapper for RTK mutation for  
 order cancellation. Returns a tuple:

- first item is a mutation callback, which accept object with `orderId` field
- second item is RTK query state
