## Types

```tsx
interface ConfirmPassword {
  confirmPassword: string;
}

interface PasswordFormData {
  password: string;
}

interface UpdatePasswordFormData extends PasswordFormData, ConfirmPassword {}
```

## Components and Widgets

### UpdatePasswordForm

This is form layout to update the password (password and confirm password fields) that is driven by
react-hook-form's `useForm`. Accepts callback and query state from `useUpdatePassword` hook (see below).

<ArgTypes of={UpdatePasswordForm} />

## Hooks

### `useUpdatePassword()`

This hook provides a wrapper for RTK mutation. Returns a tuple:

- first item is a mutation callback, which accepts `UpdatePasswordFormData` as an argument and returns a promise
- second item is RTK query state
